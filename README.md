## Описание
Приложение для управления личной коллекцией фильмов. Позволяет добавлять фильмы с указанием названия, жанра, года и рейтинга, а также фильтровать их по жанру и году выпуска. Все данные сохраняются в файл `JSON`.

**Автор:** Сабирзянов Марсель
**Дата:** 05.05.2026

## Функциональность
- **Добавление фильмов:** Ввод данных с валидацией (год - число, рейтинг 0-10).
- **Табличный просмотр:** Отображение всех фильмов в удобной таблице.
- **Фильтрация:** Поиск фильмов по жанру и диапазону годов.
- **Сохранение:** Автоматическое сохранение данных в `movies.json`.
